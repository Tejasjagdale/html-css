Preloader Tutorial using CSS3

Shivam Deotarse

   22 OCT 2020

   6 MIN READ

HTML
CSS
PRELOADER

   LANGUAGES

HTML  20%

CSS  60%

JavaScript  20%

Today’s tutorial is about creating an amazing preloader with CSS3 animations which is used for showing loading screens. The idea is to create a preloader from scratch using animations and rotation transforms .Our preloader has a rotating pen and some ink to show loading time.

TOPICS

  • Introduction
  • HTML

  • CSS

  • JAVASCRIPT

THE HTML

For markup we will have main div 'line' used to draw line. And then we have div 'pen' for rotating pen. We also have div 'ink' for showing animations of ink filled and removed from pen.

So basically we have created a div for all different elements like pen, ink, line etc.

Let’s look at our style.

THE CSS

First we begin with setting padding and margin of whole document to 0.

Next we style our body.Setting background-color 'black' and centering our div 'line' with justify-content and align-items properties.

Now we set width and height of 'line' div with 'relative' position.

Next step is to add a mask of black color over 'line' div using 'line:before'. In 'line:before' we have used color 'black' and used an animation 'draw' with 'linear' effect for '6 sec'.

Now we will be defining 'draw' animation using '@keyframes' where we divide time for which animation occurs in percentages and accordingly we set width of mask we are adding using 'line:before'.

Let's work on our 'pen' div and see how we are rotating our 'pen' div. Firstly setting position of 'pen' div to absolute and placing the div using top, right and left properties. Using transform property rotating our 'pen' div to initial orientation. And we have defined animation 'rps' for rotating our pen by dividing time in percentage and using transform property to 'rotate' 'pen' div at different intervals of time.

Next we design the triangular tip of pen using 'pen:before'. Triangle shape is due to "border-color: transparent transparent transparent orange;
border-width: 20px 40px 21px;".
Here we divide rectangular 'line:before' div with diagonals into 4 triangles and set 3 triangles as transparent.

With 'pen:after' div we design the smaller white triangular nip of the pen in similar way as that of 'pen:before' div.

Now we fill our 'pen' div with some orange ink.We will be using 'ink' animation using '@keyframes'.

In our 'ink' animation we set width of 'ink' div according to percentage interval of total animation time.

Then using 'eraser' div we design our eraser of the pen by setting background-color as 'orange' and border properties.

Here our style code ends completing awesome tutorial for our creative preloader ditching boring loading spinners. Peace.
