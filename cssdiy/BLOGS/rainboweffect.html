Today’s tutorial is about creating an amazing button effect with CSS3 in which the rainbow colors move in the button.

this is suerly a good effect to add if you want your website to look attractive.

THE HTML

The HTML part is very simple we have to take a button under a div.

The html part of this effect is nearly nothing most work is of css3.

THE CSS

first set the button to center by display:flex then set the background color for buttons parent div

then comes the important css part which is to style the button here basically the key point is we are gona set a linear-gradient and then make it move now how it will happen lets see.

so set proper width and height of button now set some imp properties also such as outline:none,text-decoration:none; and box-sizing:border-box

now set background-image:linear-gradient(90deg,#03a9f4,#f441a5,#ffeb3b,pink,#03a9f4); then give background-size:500%; and now we will set animation property to animation:animate 10s linear infinite;

our animation is now on last set as we have set the background to 500% and at a time only 1005 can be visiable so we will set background-position to 0% and then animate it to 500% so it will keep on moving 0% to 500% as we have set iterration to infinite.
