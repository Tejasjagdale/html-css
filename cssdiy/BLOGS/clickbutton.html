click button Tutorial using CSS3

   22 OCT 2020

   6 MIN READ

Today’s tutorial is about creating an amazing BUTTON with CSS3 animations which will surely add value to your website or projects. The button has a ripple effect so when we click, the color of button keeps on changing with amazing transition.

TOPICS

  • Introduction
  • HTML

  • CSS

  • JAVASCRIPT

THE HTML

The markup part of this effect is very basic. It consists of a button within a div which will be centered afterwards and a para tag

So basically the most work is of CSS and JavaScript in this.

Let’s look at our style.

THE CSS

First we begin with setting padding and margin of whole document to 0.

Next we style our body. Setting background-color 'black' or we can take any other background-color we want but it should not match with button color.

Now we set the div and button to center with display property as 'flex' and set justify-content and align-items properties 'center'

Next step is to set the para tag above the button with position absolute we will come to it why we are taking a extra para tag so just set proper left and top distance and bring it to top of button

now set the default color to button and para text. and set some border-radius too you can also give border too

you can refer the images if you have any doubt.as there is some more css but which we make sense after a littel javascript part so we will disscus it there.

The css part is done now lets jump to final part i.e javascript.

THE JAVASCRIPT

In this effect the most work is of javascript.

so first we set a variable i with value 0 which has important role afterwards. Then we access the button by DOM and. set addEventListener() function and set event as click and pass event in function by writing it like this [function(e)] here "e" is just used so that our event handler can refer back to the object and set and increment with i++.

now as we can see in button effect it is direction aware it knows were the cursor is clicked on button

so to achive that we get the buttons distance form left side by e.clientX and same way distance from top by e.clientY. now we need the position where the cursor is clicked on button we get it by e.target.offsetLeft and e.target.offsetTop basically they give the position of cursor from left and top now if we subtract both we will get the relative position of cursor so now we know where the cursor click exactly on button

you can check it by console.log(x) and console.log(y). so now its time to decide which colors we want on button so make a array and keep first elemnt "0" and then add as much color you want which ever colors you like but remember the number of colors you have given as we will require that info.

now we get to our ripples effect so we use here the createElement function and create span tag dynamicaly under button tag and here comes again use of css as i told you so we give the button position relative and span absolute so as we have now the cursor position and power to place span tag anywhere in button we place it in the position where user clicks. and style it as in image and we will keep on changing its position as we add variable to left and top in setAttribute

now set background-color of span also variable so that it change on every click and also keep width and height of span more than button and button on overflow hidden and we set para tag above button bescause that then span tag should not come above para tag

now comes the final part i.e setTimeout function part in which we set timeout equal to transition time off span tag as we are gona remove the created span tag and gona acctuly change the color of button so at same time the span tag dissappers and color of button changes. also there comes a problem of i as it will keep on increasing so to avoid that we try make a loop by adding if condition in which we will give if (i) is more than number of colors we given in array then make it zero again.

Thanks for beign patient and reading till here hope you get this effect and it may be usefull for you.
